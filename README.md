# Fourty-two

Example flask app that serves a fun "42-digit" webpage.
Database is a simple `digits` file - no tables/sql.

## How to use

Clone this repo. Ensure flask is installed:
```bat
python -m pip install flask
```
`cd` to the root and run
```bat
flask run --debug
```
